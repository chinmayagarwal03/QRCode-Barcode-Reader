# QRCode-Barcode-Reader

You can easily read any type of QR code or barcode with the help of this project.

### Libraries used
1. Numpy
2. OpenCV
3. Pyzbar - Read one-dimensional barcodes and QR codes from Python 2 and 3 using the zbar library. 

For Installation ~~ 

brew install zbar 

pip install pyzbar
